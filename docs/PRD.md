# LifeOS - Product Requirements Document

## Overview

LifeOS is an AI-powered productivity platform that helps users manage tasks, build habits, track goals, and improve personal discipline.

## Goals

* Improve productivity
* Build consistency
* Track habits
* Organize daily life
* Provide AI-powered recommendations

## MVP Features

* Authentication
* Task Management
* Habit Tracking
* Dashboard

## Future Features

* Pomodoro Timer
* Notifications
* Analytics
* AI Coach
* Goal Planner
* Mobile App

## Tech Stack

Frontend:

* Next.js
* TypeScript
* Tailwind CSS
* Shadcn UI

Backend:

* NestJS
* Prisma

Database:

* PostgreSQL
